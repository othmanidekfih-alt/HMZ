## 🚀 Step-by-Step Instructions

### Step 1: Create Vercel Account (2 minutes)

1. Open your phone browser
2. Go to: **https://vercel.com/signup**
3. Click **"Continue with GitHub"**
4. Log in with your GitHub account (othmanidekfih-alt)
5. Allow Vercel to access your repositories
6. ✅ Done!

---

### Step 2: Deploy Your App (3 minutes)

1. On Vercel, click **"Add New..."** → **"Project"**
2. Find your repository: **"HMZ"**
3. Click **"Import"**
4. Vercel will detect it's a Vite app automatically
5. **IMPORTANT**: Set the Root Directory to `rabat-discover`
   - Click "Edit" next to Root Directory
   - Type: `rabat-discover`
6. Leave everything else as default
7. Click **"Deploy"** button
8. Wait 1-2 minutes while it builds...
9. 🎉 **You'll get a live link!**

## 🤔 If You Get Stuck

### "Vercel can't find my repository"
- Make sure you're logged in with the right GitHub account
- Refresh the page
- Try clicking "Adjust GitHub App Permissions"

### "Build failed"
- Check if Root Directory is set to `rabat-discover`
- Screenshot the error and show me
- I'll fix it!

### "App loads but map doesn't show"
- Allow location permissions
- Try on WiFi (not mobile data)
- Clear your browser cache

## 📱 Installing as an App

Once deployed, you can "install" it:

**On iPhone:**
1. Open the link in Safari
2. Tap the Share button
3. Scroll down → "Add to Home Screen"
4. Tap "Add"
5. Now it's an app icon! 🎉

**On Android:**
1. Open the link in Chrome
2. Tap the menu (⋮)
3. Tap "Add to Home screen"
4. Confirm
5. Now it's an app icon! 🎉
